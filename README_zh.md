# ftidy

文件整理工具：去重和批量重命名。

## 安装

```sh
cargo install --path .
```

## 使用

### dedup — 查找并删除重复文件

```sh
# 预览模式：显示重复文件但不删除
ftidy dedup ~/Photos

# 实际删除重复文件（会交互确认）
ftidy dedup ~/Photos --delete
```

通过 SHA-256 内容哈希比较文件。保留首次出现的文件，报告（或删除）其余副本。

特性：
- 先按文件大小分组，仅对大小相同的文件计算哈希（减少 I/O）
- 流式哈希，不会因大文件耗尽内存
- 跳过符号链接和空文件
- 删除前需交互确认

### rename — 批量重命名文件

```sh
ftidy rename *.jpg -p "vacation_{n}"
```

模板变量：
- `{n}` — 序号（001, 002, ...）
- `{name}` — 原文件名（不含扩展名）
- `{ext}` — 原扩展名

如果模板中不含 `.`，则自动保留原扩展名。

特性：
- 执行前检测目标文件冲突
- 失败时自动回滚已重命名的文件
